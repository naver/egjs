GUIDE: Downloading and Using egjs

Download

  • eg.js un-minified
  • eg.min.js minified
  • eg.pkgd.min.js (with every dependency except jQuery)
  • flicking.pkgd.min.js
  • infiniteGrid.pkgd.min.js

CDN

Link directly to egjs files on cdnjs

<script src="https://cdnjs.cloudflare.com/ajax/libs/egjs/1.3.0/pkgd/eg.pkgd.min.js"></script>

Package managers

$ npm install egjs --save
$ bower install egjs --save
